Essential HTML Tags

1. <html> - Root Element

<html lang=\"en\">

2. <head> and <body>

<head> meta info </head> <body> visible content </body>

3. <h1> to <h6> - Headings

<h1>Main Title</h1>

4. <p> - Paragraph

<p>This is a paragraph.</p>

5. <a href> - Hyperlink

<a href=\"https://example.com\">Visit</a>

6. <img> - Image

<img src=\"image.jpg\" alt=\"desc\" />

7. <ul>, <ol>, <li> - Lists

<ul><li>Item</li></ul>

8. <div> and <span>

<div>Block</div> <span>Inline</span>

9. <table>, <tr>, <td>

<table><tr><td>Data</td></tr></table>

10. <form> and Input Fields

<form><input type=\"text\" /></form>

11. <button>

<button>Click</button>

12. <br> and <hr>

<br> line break <hr> horizontal rule

13. <meta>

<meta charset=\"UTF-8\">

14. <link> - Stylesheet

<link rel=\"stylesheet\" href=\"style.css\">

15. <script> - JavaScript

<script src=\"script.js\"></script>
